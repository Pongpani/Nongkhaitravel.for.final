หน้าแรก เกี่ยวกับจังหวัด ข้อมูลโครงงาน ติดต่อเรา Admin

ข้อมูลโครงงาน: เว็บไซต์แนะนำสถานที่ท่องเที่ยวอำเภอเมืองหนองคาย

โครงงานจบ ปวช.3 – สาขาคอมพิวเตอร์ธุรกิจ

1) ชื่อโครงงาน

NongkhaiTravel – เว็บไซต์แนะนำสถานที่ท่องเที่ยวอำเภอเมืองหนองคาย

2) ที่มาและความสำคัญ

หนองคายเป็นเมืองท่องเที่ยวสำคัญริมโขง แต่ข้อมูลกระจัดกระจาย เว็บไซต์นี้พัฒนาขึ้นเพื่อรวบรวมสถานที่เด่น พร้อมคำอธิบายและแผนที่ในที่เดียว ใช้งานง่าย เหมาะสำหรับนักท่องเที่ยวและผู้ที่ต้องการแนะนำเมืองให้ผู้อื่นรู้จัก

3) วัตถุประสงค์

  • พัฒนาเว็บแอปแบบสถิตสำหรับเผยแพร่ข้อมูลแหล่งท่องเที่ยว
  • ฝัง Google Maps เพื่อช่วยวางแผนการเดินทาง
  • มีหน้าหลังบ้านจำลองให้แก้ไขข้อมูลได้ด้วย LocalStorage

4) ขอบเขตของระบบ

  • หน้าผู้ใช้: หน้าแรก, เกี่ยวกับ, ติดต่อ, ข้อมูลโครงงาน
  • หลังบ้าน (จำลอง): เพิ่ม/แก้ไข/ลบสถานที่, ระบบใส่รหัสผ่านก่อนเข้า
  • ไม่มีฐานข้อมูลเซิร์ฟเวอร์จริง ใช้ LocalStorage เพื่อการสาธิต

5) เครื่องมือและเทคโนโลยีที่ใช้

  • Frontend: HTML, CSS, JavaScript (Vanilla)
  • Data: LocalStorage, โครงสร้างข้อมูล JSON
  • Map: Google Maps Embed
  • โฮสต์: GitHub Pages

6) โครงสร้างไฟล์

/
├─ index.html
├─ about.html
├─ project.html
├─ contact.html
├─ login.html
├─ admin.html
└─ assets/
   ├─ styles.css
   ├─ data.js
   └─ app.js

7) คู่มือการใช้งานโดยย่อ

  1. เปิด index.html เพื่อดูรายการสถานที่
  2. กด เปิดพรีวิวแผนที่ เพื่อดูแผนที่ฝัง
  3. เข้าสู่ระบบ Admin ผ่านหน้า login.html (รหัสเริ่มต้น: admin123)
  4. เพิ่ม/แก้ไข/ลบสถานที่ แล้วข้อมูลจะถูกบันทึกในเบราว์เซอร์

8) แผนพัฒนาต่อ

  • ย้ายข้อมูลจาก LocalStorage ไปสู่ฐานข้อมูลจริง (เช่น Firebase)
  • เพิ่มระบบอัปโหลดรูปภาพ
  • รองรับหลายอำเภอ/หลายภาษา

9) ผู้จัดทำ

วรงค์กร นันทมีชัย (Bas) – วิทยาลัยการอาชีพหนองคาย, สาขาคอมพิวเตอร์ธุรกิจ
