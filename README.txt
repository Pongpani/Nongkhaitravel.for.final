NongkhaiTravel (ภาษาไทยล้วน) – รองรับหลายรูปต่อสถานที่
------------------------------------------------
วิธีใช้งาน:
1) แตกไฟล์โฟลเดอร์ NongkhaiTravel_TH_MultiImage แล้วเปิดไฟล์ index.html ด้วยเบราว์เซอร์
2) หน้าแอดมิน: ไปที่ login.html รหัสผ่านเริ่มต้น: admin123 (แก้ได้ใน login.html)
3) หากต้องการใส่หลายรูป ให้กรอกลิงก์ภาพคั่นด้วยเครื่องหมายจุลภาค (,) เช่น
   https://a.jpg, https://b.jpg, https://c.jpg
4) หน้ารายละเอียด (place.html) จะ:
   - แสดงเป็นแกลเลอรีแบบกริดอัตโนมัติ หากมีมากกว่า 1 รูป
   - แสดงภาพเดียวใหญ่ หากมีรูปเดียว

ไฟล์สำคัญ:
- index.html / place.html / about.html / contact.html / login.html / admin.html
- assets/styles.css / assets/app.js / assets/data.js
